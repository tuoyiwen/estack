# Anthropic Claude Web Search Tool

## Archive Status

Source card only. Official documentation is linked, not copied.

## estack Relevance

Anthropic frames web search as access to current web content with cited sources, optional dynamic filtering, and domain controls. For estack, this supports source control, allowed/blocked domains, dynamic filtering, and admin governance for enterprise search workflows.

## Extracted Use

- Add domain allow/block controls to governance checklists.
- Treat dynamic filtering as a way to manage context quality before synthesis.
- Record cost / token implications for repeated searches in enterprise workflows.
