# Source Card: Anthropic, "Effective Context Engineering for AI Agents"

- Date: 2025-09-29
- Status: 官方文档，未全文归档

## 核心主张

Context engineering 的正式定义：策划和维护最优 token 集（信息）的策略集，包括 prompt 之外落入 context 的所有信息。Context window 是有限资源，token 数量增加会导致召回准确率下降（Context Rot）。

## estack 用途

L2 层的权威定义出处；token 成本治理中"最小高信号 token 集"原则的一手来源。
