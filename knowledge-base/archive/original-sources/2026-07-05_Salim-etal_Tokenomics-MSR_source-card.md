# Source Card: Salim et al. (2026), "Tokenomics: Quantifying Where Tokens Are Used in Agentic Software Engineering"

- Authors: Salim, M., Latendresse, J., Khatoonabadi, S. H., & Shihab, E.
- Venue: Mining Software Repositories (MSR) 2026
- Status: 未取全文，经腾讯研究院 Token 不经济一文二手引用

## 核心发现

对 ChatDev 多 Agent 框架 30 个任务的分析：Code Review 阶段平均消耗总 token 的 39.5%，为所有开发阶段最高——近四成 token 花在 Agent 之间反复传递已有信息（沟通税 communication tax），而非生成新内容。

## estack 用途

多 Agent 沟通税的实证数据；`$e-innovation-metrics` 无效消耗占比指标的依据。学术论文，evidence strength: direct（引用前建议核对原文数字）。
